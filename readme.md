Laravel com Vue.js
----------

Repositório criado para um curso da Udemy.
quem se interessou é o curso Laravel com Vue.

**Requisitos:**
-  Composer;
-  Laravel 5.5;
-  PHP 7.0;
-  Node 8.1.2;

in a bash:

1 - ```composer install ```;
2 - ```npm install ```;
3 - ```cp .env.example .env ```;
4 - ```php artisan key:generate ```;
5 - ```php artisan serve ```;
6 - for js, vue or sass changes, run ```npm run watch ```;
